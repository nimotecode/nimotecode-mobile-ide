# AI Guide

NimoteCode includes AI Chat and AI Agent workflows.

## AI Chat

Use AI Chat for:

- code explanation
- bug analysis
- refactor suggestions
- terminal output analysis

## AI Agent

AI Agent provides tool-driven execution with runtime feedback, including:

- execution steps
- progress and status
- risk/warning hints
- auto mode / confirm mode

## Provider Configuration

Built-in templates:

- OpenAI
- Anthropic
- Google Gemini
- xAI Grok
- DeepSeek
- Zhipu AI
- Moonshot AI
- Ollama
- Custom OpenAI-compatible

Configurable fields include:

- display name
- provider key
- API key
- base URL
- model
- temperature
- max tokens
- top-p
- streaming toggle
- system prompt
- context window size
- file context toggle

## Provider Selection

- AI Chat and Git AI workflows can select provider configs.
- Ensure at least one valid provider config is saved.
