# Source Control Workflow

This page documents the in-app Source Control (Git) workflow.

## Repository View

The Source Control panel shows:

- staged changes
- unstaged changes
- conflicts
- branch context

## Core Actions

- stage / unstage file changes
- discard file changes (with confirmation)
- stage all / unstage all / discard all
- open diff views

## Commit and Sync

- enter commit message
- commit changes
- pull / push / fetch operations
- optional amend last commit

## Branch Operations

- create branch
- checkout branch
- merge branch
- rebase branch
- rename branch
- delete branch
- compare with current branch

## Stash Operations

- create stash (optional message, include untracked)
- apply stash
- pop stash
- drop stash
- clear all stashes

## History and Diff

- commit history
- file history
- line history
- diff pages

## AI for Git

Integrated Git AI tools include:

- generate commit message
- explain file changes
- analyze change risks
