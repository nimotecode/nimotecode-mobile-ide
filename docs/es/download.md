# Descargar

NimoteCode está disponible para iOS y Android.

## App Store

[![Descargar en la App Store](https://developer.apple.com/assets/elements/badges/download-on-the-app-store.svg)](https://apps.apple.com/app/nimotecode)

## Google Play

[![Obtener en Google Play](https://play.google.com/intl/en_us/badges/static/images/badges/en_badge_web_generic.png)](https://play.google.com/store/apps/details?id=com.nimote.nimotecode)

## Requisitos del sistema

- **iOS**: iOS 15.0 o posterior
- **Android**: Android 8.0 (nivel de API 26) o posterior
- **Recomendado**: 4 GB de RAM o más, 64 GB de almacenamiento o más
