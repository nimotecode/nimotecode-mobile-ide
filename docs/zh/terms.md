# 服务条款

**生效日期：** 2026-04-16

本条款适用于 NimoteCode 的使用。

## 1. 服务范围

NimoteCode 提供移动端 IDE 能力，包括本地/远程工作区编辑、终端工作流、Source Control 工作流、AI 配置与集成、任务执行和时间线视图。

## 2. 账户与访问

- 请妥善保管账户凭证与设备安全。
- 账户下发生的操作由账户持有人负责。

## 3. 订阅

- 部分功能仅对 Pro 订阅用户开放。
- 购买与续费由应用商店平台处理。

## 4. 合规使用

您不得：

- 未经授权访问系统
- 利用应用进行违法活动
- 逆向工程或滥用服务

## 5. 第三方服务

您可自行配置 AI 提供商与远程服务器。相关凭据管理、数据传输与第三方条款合规由您负责。

## 6. 责任说明

在法律允许范围内，服务按“现状”提供。

## 7. 条款更新

我们可更新条款。更新后继续使用即表示接受新条款。
