# 功能特性

以下内容与当前实现保持一致。

## 工作区

- Local Explorer 模式
- Remote Explorer（SSH）模式
- 工作区/会话历史
- Sync / Cache 面板流程

## 编辑器

- 语法高亮
- 符号大纲
- 文件内查找/替换
- 符号搜索
- 基础文件操作（打开、保存、重命名、删除）

## 终端

- 多终端标签
- 快捷命令面板
- 终端输出搜索（基础/高级）
- 终端输出发送至 AI 分析

## Source Control

- 仓库状态（已暂存/未暂存/冲突）
- 暂存/取消暂存/丢弃
- 提交与同步流程
- 分支操作（创建、切换、合并、变基、重命名、删除）
- Stash 操作（创建、应用、弹出、删除、清空）
- 提交历史、文件历史、行历史、Diff 页面
- Git AI（提交信息生成、改动解释、风险分析）

## AI

- AI Chat
- AI Agent（步骤、进度、风险提示、自动/确认模式）
- 提供商高级参数：temperature、max tokens、top-p、streaming、system prompt、context window、file context

内置模板：

- OpenAI、Anthropic、Google Gemini、xAI Grok
- DeepSeek、Zhipu AI、Moonshot AI
- Ollama
- Custom OpenAI-compatible

## Tasks

- Tasks 面板
- 任务分组、作用域、执行配置
- 任务编辑与运行

## Timeline

- 事件列表
- Trace 视图
- 因果分析流程状态
- 过滤与清理

## 登录与订阅

- 邮箱登录
- Google 登录
- 订阅校验与 Pro 功能门控
