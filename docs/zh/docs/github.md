# Source Control 工作流

本文档对应应用内 Source Control（Git）能力。

## 仓库视图

面板可查看：

- 已暂存变更
- 未暂存变更
- 冲突
- 当前分支上下文

## 核心操作

- 暂存 / 取消暂存 / 丢弃变更
- 全量暂存 / 取消暂存 / 丢弃
- 打开 Diff 对比

## 提交与同步

- 输入提交信息
- 提交变更
- pull / push / fetch
- 可选 amend 上次提交

## 分支操作

- 创建分支
- 切换分支
- 合并分支
- 变基分支
- 重命名分支
- 删除分支
- 与当前分支比较

## Stash 操作

- 创建 stash（可选消息，可包含未跟踪文件）
- 应用 / 弹出 / 删除 stash
- 清空 stash

## 历史与 AI

- 提交历史、文件历史、行历史、Diff 页面
- Git AI：提交信息生成、改动解释、风险分析
