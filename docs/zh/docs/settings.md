# 设置

设置页帮助你把 NimoteCode 调整为稳定、舒适、长期可用的移动开发环境。

## 账户与订阅

- 登录与账户身份管理
- 订阅状态与试用信息查看
- 订阅管理与恢复购买入口

## 外观与语言

- 主题模式切换
- IDE 主题风格选择
- 多语言切换

## 编辑体验

- 字号与行高调节
- 字体族偏好
- 跨会话持久化生效

## 键盘与输入

- 软键盘开关
- 触觉与按键音反馈
- 数字行与功能键行开关
- 手势与功能键映射
- 键盘主题与一键重置

## 政策入口

设置页可直接访问：

- [账号删除](/zh/account-delete)
- [隐私政策](/zh/privacy)
- [服务条款](/zh/terms)

## 推荐首次设置

1. 先确定语言与主题。
2. 调整编辑器排版参数。
3. 配置键盘行为以匹配设备习惯。
4. 检查账户与订阅状态。
