# AI 使用指南

NimoteCode 提供 AI Chat 与 AI Agent 两类能力。

## AI Chat

适用于：

- 代码解释
- 错误分析
- 重构建议
- 终端输出解读

## AI Agent

Agent 提供工具驱动执行流，包含：

- 步骤反馈
- 进度与状态
- 风险提示
- 自动模式 / 确认模式

## 提供商配置

内置模板：

- OpenAI
- Anthropic
- Google Gemini
- xAI Grok
- DeepSeek
- Zhipu AI
- Moonshot AI
- Ollama
- Custom OpenAI-compatible

可配置参数：

- display name
- provider key
- API key
- base URL
- model
- temperature
- max tokens
- top-p
- streaming
- system prompt
- context window size
- file context
