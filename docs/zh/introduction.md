# 简介

NimoteCode 是面向本地与 SSH 远程工作区的移动端 IDE。

## 你可以做什么

- 打开本地目录或连接 SSH 远程工作区
- 在编辑器中进行语法高亮编辑、查找替换、符号导航
- 在多标签终端中运行命令
- 在 Source Control 面板管理 Git 变更
- 配置 AI 提供商并使用 AI Chat / AI Agent
- 在 Tasks 面板运行任务，在 Timeline 面板查看事件与追踪

## 技术栈

- 前端：Flutter (Dart)
- 插件/原生层：Rust（通过 Flutter Rust Bridge）

## 推荐工作流

1. 选择本地或远程工作区
2. 编辑并保存代码
3. 使用终端执行命令
4. 在 Source Control 中提交变更
5. 使用 AI 辅助解释、重构和自动化处理
