# 账号删除

**最后更新：** 2026-04-16

本页说明如何删除 NimoteCode 账号及关联云端数据。

## 应用内删除（推荐）

若你仍可登录应用：

1. 打开 **设置**。
2. 点击 **删除账号**。
3. 确认删除。

确认后，NimoteCode 账号及关联云端用户记录将被删除。

## 网页删除请求（无法登录时）

若你无法登录应用，可通过以下渠道提交删除请求：

- 邮箱：aoun.ma@foxmail.com
- GitHub Issues：https://github.com/nimotecode/nimote_issues/issues

请尽量提供：

- 账户邮箱（如有）
- 平台（iOS/Android）
- 请求类型：`账号删除`

## 删除范围

删除将覆盖 NimoteCode 账号数据及用于账户/订阅状态的关联云端记录。

本地工作区文件、远程服务器文件，以及你自有基础设施上的数据由你自行控制，不会因账号删除自动清除。

## 处理时效

在完成身份核验后，我们会在合理时间内处理删除请求；但可能因法律义务、安全防滥用、争议处理等原因保留必要记录。
