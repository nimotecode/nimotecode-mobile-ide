# Features

This page reflects the current implemented feature set.

## Workspace

- Local Explorer mode
- Remote Explorer mode (SSH)
- Session history for explorer/workspace access
- Sync / Cache panel workflows

## Editor

- Syntax highlighting
- Symbol outline panel
- Find / replace in file
- Symbol search
- Basic file actions (open, save, rename, delete)

## Terminal

- Multi-terminal tabs
- Quick commands panel
- Terminal output search (basic and advanced)
- Send terminal output to AI for analysis

## Source Control

- Repository status view (staged/unstaged/conflicts)
- Stage / unstage / discard actions
- Commit and push flows
- Branch actions (create, checkout, merge, rebase, rename, delete)
- Stash actions (create, apply, pop, drop, clear)
- Commit history, file history, line history, and diff pages
- AI-assisted commit message / change explanation / risk analysis

## AI

- AI Chat panel
- AI Agent execution flow with step/risk/progress feedback
- Provider configuration with advanced params:
  - temperature, max tokens, top-p
  - streaming toggle
  - custom system prompt
  - context window size
  - file context injection toggle

Supported provider templates:

- OpenAI, Anthropic, Google Gemini, xAI Grok
- DeepSeek, Zhipu AI, Moonshot AI
- Ollama
- Custom OpenAI-compatible endpoint

## Tasks

- Tasks panel
- Task groups / scope / execution settings
- Task editor dialog
- Run tasks and stream output to terminal

## Timeline

- Event list view
- Trace view
- Causal analysis workflow states
- Filter and clear actions

## Login / Subscription

- Email + password login
- Google sign-in flow
- Subscription and premium access checks
- Pro feature gating for selected capabilities
