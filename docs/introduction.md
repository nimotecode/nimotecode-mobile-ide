# NimoteCode Introduction

**AI-Powered Mobile SSH IDE for Real Development Workflows**

NimoteCode is a mobile SSH IDE for developers and ops engineers. It lets you do real development, debugging, and delivery work from your phone or tablet.

It is not an editor. It is a complete **remote development + AI-driven engineering workspace**.

## Why Teams Choose NimoteCode

- One workspace that combines **Code Editor / Terminal / Git / AI / Debug**
- Built for real **SSH remote development workflows** in production environments
- Deeply integrated **LSP, Debug, Tasks, and Timeline** tools for faster issue resolution
- Built for mobile productivity with multi-panel workflows, keyboard handling, and tablet-friendly layouts

## AI-Driven Development Experience

- AI Code Explain: understand complex code quickly
- AI Refactor: improve and restructure code automatically
- AI Fix (LSP): use diagnostics to repair issues
- AI Terminal: understand and generate shell commands
- AI Timeline Analysis: trace code changes and problem origins
- AI Agent: execute multi-file development and ops tasks

## What You Can Do

- SSH into remote servers or local workspaces and develop directly
- Run build / test / deploy commands in multi-tab terminal sessions
- Use Git for commits, branching, code review, and sync
- Handle daily development and emergency hotfixes from mobile
- Automate repetitive engineering and troubleshooting tasks with AI

## Built for Real Production Scenarios

NimoteCode is designed for real engineering environments, not learning or toy workflows:

- Emergency hotfixes for live incidents
- Fast on-call troubleshooting
- Remote development and code submission from mobile
- Automation and script execution for operations
- Debugging distributed systems and production issues
