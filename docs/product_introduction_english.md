# NimoteCode Product Introduction

【One-sentence Positioning】
Your pocket AI coding workspace that lets you complete the entire development workflow from coding to deployment right on your phone

【What Problems We Solve】
- Need to fix code but your computer isn't with you
- Maintaining remote servers requires juggling multiple tools
- Stuck troubleshooting issues for hours
- Can't remember complex commands
- Disjointed development process that's hard to trace back
- Switching between multiple tools is cumbersome and inefficient
- Want to learn in spare time but limited by environment

【Typical Use Cases】
1. **Emergency Fixes During Commute**  
   Get an alert on the subway, pull out your phone, connect to the server, locate and fix the issue in minutes

2. **Rapid Prototyping at Coffee Shops**  
   Inspiration strikes? Use AI assistant to generate code frameworks directly, writing and verifying in terminal as you go

3. **Server Maintenance from Your Couch**  
   Check logs, update configurations, deploy code from home—no need to open your computer

4. **Timely Team Collaboration Response**  
   Review code changes, merge branches anytime without slowing down project progress

5. **Problem Retrospective & Continuous Learning**  
   Review your entire development process through the timeline, AI helps analyze what went wrong and how to avoid it next time

【What You Can Do】
- Write code anytime, anywhere with syntax highlighting for multiple languages
- Open multiple files simultaneously and switch between them easily
- Connect to remote servers with one tap, edit files and run commands directly
- Open multiple terminal sessions in parallel to handle multiple tasks
- Save frequently used commands for quick one-tap execution
- Quickly search and find content in terminal output
- Visually manage code versions, view changes, commit code
- View file and line modification history to see who changed what and when
- Help resolve code conflicts during team collaboration
- Let AI write code, explain code, and even modify files automatically for you
- Search specific text content in code for quick location
- Quickly find files by name
- Search for code symbols like functions, variables, and classes
- View code structure and quickly navigate to functions or classes
- Manage development tasks and track completion status
- Record your entire development process and review through timeline
- AI helps you analyze problems in development process
- Debug code, set breakpoints and inspect variables
- Sync project files between local and remote
- Manage multiple SSH connections, securely save login credentials
- Switch between different interface themes, choose your preferred style
- Configure keyboard-related preferences
- Choose different AI models, customize AI experience
- Save and use code snippets, quickly insert commonly used code
- Quickly log in to account, support multiple login methods
- Receive app notifications, stay informed about important information

- Full-stack developers who like to brainstorm and validate ideas on the go
